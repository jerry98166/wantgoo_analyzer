# 玩股網 x 台股大數據：旗艦版量化籌碼分析系統 (Vue 3 + FastAPI)

本專案利用 Vue 3 作為前端框架，搭配 FastAPI 作為後端 API，打造出媲美專業法人機構的「台股量化籌碼分析終端機」。系統整合了台股每日交易數據、三大法人買賣超、信用交易模擬以及玩股網的市場情緒文章，提供全方位的深度解析。

## 🌟 核心功能與亮點

### 1. 全市場大局觀
*   **戰情室儀表板 (Dashboard)**：首頁速覽當日大盤多空氛圍、三大法人總淨買賣、熱門新聞與當日最吸金權值股。
*   **大盤多空結構**：追蹤每日上漲/下跌家數比例，判斷市場真實廣度。
*   **產業板塊資金熱力圖 (Treemap)**：透過視覺化的熱力圖，一眼看穿資金正在湧入或撤出哪些產業板塊。
*   **吸金排行榜 (Turnover Leaders)**：計算個股「成交金額」，找出市場真正的資金聚寶盆。

### 2. 法人與散戶動向深度拆解
*   **土洋同步與對作**：交叉比對「外資」與「投信」，找出雙方齊買的飆股，或是互相對作的分歧股。
*   **特定法人雷達 (投信/自營商)**：
    *   **投信作帳雷達**：掃描投信高參與度標的，抓出季底作帳行情。
    *   **自營避雷針**：掃描自營商異常大買標的，提防隔日沖倒貨風險。
*   **散戶 vs 法人對作 (Retail Sentiment)**：利用融資增減數據，找出「散戶停損、法人吃貨」的絕佳買點，或是「散戶接刀、法人出貨」的危險陷阱。

### 3. 實戰選股與熱點追蹤
*   **AI 多條件選股器 (Stock Screener)**：提供股價、漲跌幅、法人買超、成交量等多維度篩選，並支援 **一鍵匯出 CSV**。
*   **聰明錢偷偷吃貨 (Smart Money)**：獨家演算法，掃描「成交量小 + 股價未動 + 法人爆買」的潛力伏兵。
*   **當沖與短線熱點 (Day Trading)**：找出當日高振幅（上衝下洗大）與跳空開高的標的，提供短線當沖客最佳戰場。
*   **籌碼與股價關聯 3D 氣泡圖**：結合漲跌幅、買賣超與成交量（氣泡大小），立體呈現強勢股輪廓。

### 4. 專業個股籌碼分析 (全方位技術面)
*   **多層次專業 K 線圖 (Candlestick)**：提供真實的開高低收 K 線圖，並附帶 5日(MA5)、20日(MA20)、60日(MA60) 均線。
*   **法人波段預估成本線**：獨家計算法人近 60 日的加權買入成本，直接畫在 K 線圖上作為關鍵防守線。
*   **技術指標連動**：下方同步呈現「法人買賣超柱狀圖」、「RSI 相對強弱指標」、「MACD 趨勢指標」，支援時間軸自由縮放 (DataZoom)。

## 🚀 安裝與執行 (前後端分離)

### 1. 啟動後端 API (FastAPI)
打開第一個終端機：
```bash
cd backend
pip install -r requirements.txt
python main.py
```
*(後端將運行於 `http://127.0.0.1:8000`)*

### 2. 啟動前端介面 (Vue 3)
打開第二個終端機：
```bash
cd frontend
npm install
npm run dev
```
*(前端將運行於 `http://localhost:5173`，請在瀏覽器開啟該網址)*

## 🛠 技術架構
*   **前端**：`Vue 3`, `Vite`, `Element Plus` (UI 組件庫), `ECharts` (多層次 K 線、Treemap、3D 氣泡圖)。
*   **後端**：`FastAPI`, `Pandas`, `requests`, `BeautifulSoup`, `yfinance`。
